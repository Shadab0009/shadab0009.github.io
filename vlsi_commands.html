VLSI Tool Command Reference

Comprehensive list of commands for a Cadence Timing/Physical Design Tool

Analysis

  • ::Imm::launch views module interconnection matrix

    Launch the views module interconnection matrix GUI.

  • all_connected

    Reports the list of all connected objects of the specified object.

  • check_design

    Reports information on design state.

  • check_timing_intent

    Print a timing lint report.

  • clock_ports

    Finds clock ports of the design.

  • create_timing_bin

    Create a timing bin.

  • fanin

    Traces fanin from a pin or port.

  • fanout

    Traces fanout from a pin or port.

  • report

    Generates one of various reports.

  • report_constraint

    Print a constraint report.

  • report_timing_summary

    Generate a timing report.

  • time_info

    Report and track runtime and memory performance.

  • validate_timing

    Generates a Tempus timing report.

Applets

  • applet

    Main command to install/manage/update 'applets'.

Attribute

  • report_obj

    Reports object's attribute information.

Constraint (Timing & Exceptions)

  • read_sdf

    Read annotated delays (SDF format).

  • clock_uncertainty

    Specifies the uncertainty on the clock network.

  • create_mode

    Create a new constraint mode.

  • define_clock

    Defines and apply a clock waveform.

  • define_cost_group

    Defines a new goal for timing optimization.

  • derive_environment

    Derives the environment of an instance.

  • external_delay

    Specifies delay that is outside the design.

  • multi_cycle

    Overrides default clock edge selection.

  • path_adjust

    Adjusts path constraints for timing analysis.

  • path_delay

    Constrains certain paths for timing analysis.

  • path_disable

    Disables timing analysis for certain paths.

  • path_group

    Assigns certain paths to a cost group.

  • reset_path_adjust

    Reset path adjust constraints for timing analysis.

  • specify_paths

    Describes a set of timing paths for a timing exception or a report.

  • create_base_cell_set

    Create base_cell set.

Design Manipulation

  • create_inst

    Creates a new instance of a design, subdesign or libcell.

  • insert_buffer

    Insert a buffer at given pin.

  • ungroup

    Ungroups one or more instances.

  • flatten_complex_ports

    Flatten complex ports of a (sub)design.

  • connect

    Connects a list of pins/ports/subports to each other.

  • change_names

    Change names of select objects.

  • uniquify

    Eliminates sharing of subdesigns between instances.

  • edit_netlist

    Edits a gate-level design.

  • create_port

    Creates a new port on a design.

  • create_primitive

    Creates a new unmapped logic primitive.

Design for Test (DFT)

  • replace_scan

    Replaces non-scan flops which pass DFT rules by their scan equivalent flops.

  • connect_scan_chains

    Connects scan registers and segments into scan chains.

  • insert_test_compression

    Inserts compression logic.

  • check_dft_rules

    Checks for DFT rule violations.

  • write_atpg

    Describes scan chains for ATPG interface.

  • report_chains_total_wirelength

    Reports the total scan path wire-length.

  • fix_dft_violations

    Fixes DFT rule violations.

  • analyze_testability

    Performs ATPG analysis of the design.

Input and Output

  • read_db

    Read a netlist in internal database format.

  • write_db

    Write a netlist in internal database format.

  • read_hdl

    Reads in Verilog or VHDL files.

  • read_sdc

    Reads in design constraints in SDC format.

  • write_sdc

    Writes out design constraints in SDC format.

  • write_netlist

    Writes out a design or a subdesign in Verilog.

  • write_sdf

    Writes out delay information into a Standard Delay Format (SDF) file.

  • read_def

    Reads in a DEF file (Physical).

  • write_def

    Exports floorplan in DEF format (Physical).

Navigation

  • all_clocks

    Get all clocks.

  • all_registers

    Get all registers.

  • get_cells

    Get cells (instances).

  • get_pins

    Get pins.

  • current_design

    Get current design.

  • set_top_module

    Changes to the new top module.

  • fanin

    Traces fanin from a pin or port (also in Analysis).

Timing (Core SDC & OCV)

  • create_clock

    Create clock (main SDC command).

  • create_generated_clock

    Create generated clock.

  • set_input_delay

    Set input delay (SDC constraint).

  • set_output_delay

    Set output delay (SDC constraint).

  • set_false_path

    Set false path (SDC exception).

  • set_multicycle_path

    Set multicycle path (SDC exception).

  • phys_enable_ocv

    Enables the SOCV and AOCV flows.

  • set_case_analysis

    Set case analysis (for functional muxes/pins).

  • set_timing_derate

    Set timing derate (for variation modeling).

Synthesis

  • elaborate

    Elaborates previously read HDL files.

  • syn_generic

    Synthesizes the design to generic gates.

  • syn_map

    Synthesizes the design to mapped gates.

  • syn_opt

    Synthesizes the design to optimized gates.

  • retime

    Retimes the design (moves registers across combinational logic).

  • delete_buffer_tree

    Delete all the buffers and inverter pairs.

Report (Specialized)

  • report_clock_tree_structure

    Report clock tree structure (Specific to this tool).

  • report_logic_levels_histogram

    Provides the feature to get the logic level histogram.

  • report_timing_derate

    Provides the feature to get the timing derate.
